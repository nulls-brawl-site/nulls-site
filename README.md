Nulls-site - это сайт для быстрого создания .json файлов чтобы модифицировать игру null's brawl от всеми известного t.me://danyanull
**Как запустить сайт?**
1. Скачайте последний релиз сайта, перетащите файл .py и папку csv в одно место, откуда потом вы будете запускать скрипт.
2. Установите библиотеки _Pandas, Flask._
3. Запустите скрипт.
4. Оставьте окно с открытым python в фоне.
5. Перейдите по ссылке http://localhost:5000                            
**Готово!**
